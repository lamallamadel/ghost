## [1.1.2] - 2026-03-09

### 🎉 Milestone: Sovereign Governance & Zero-Trust Architecture

Ghost CLI transitions from a utility to a full **Git Governance Platform** with a sandboxed Standard Library.

### Added
- **Standard Library**: Documentation and integration of the 14 native sandboxed extensions.
- **Zero-Trust Manifest**: Formalized security posture and mental model in README.
- **Authority Anchors**: Corrected NPM metadata and repository linkage for algorithmic trust.
- **ExtensionRunner**: Reliable SDK bootstrapper for isolated process execution.

### Changed
- **Positioning**: Ghost is now explicitly positioned as a sovereign alternative to Husky + Commitlint with security-grade auditing.
- **Isolation**: All extensions now run in isolated subprocesses with explicit capabilities.

### Fixed
- **RPC Deadlocks**: Resolved bus timeouts during high-volume intent emission.
- **AI Compatibility**: Verified support for Claude 4.6 and standardized secret masking in core audit layers.
