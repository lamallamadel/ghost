# Ghost Workbench

The Visual Command Center for the Ghost CLI ecosystem.

## Phase 1: Visual Orchestration Foundation (Completed)
This phase established the link between the CLI, the extension network, and the Desktop Console.

### Features
- **Ecosystem Sync**: Automatically discovers and maps all active Ghost extensions.
- **Console Integration**: Direct RPC link to launch and manage the `ghost-desktop-extension`.
- **Unified Dashboard Protocol**: Prepares the groundwork for visual widgets and configuration forms.

### Commands
- `ghost workbench open`: Launches the visual dashboard and synchronizes state.

## Installation
The workbench extension is a native Ghost orchestrator.
```bash
ghost extension install extensions/ghost-workbench-extension
```
